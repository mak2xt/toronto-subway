# Synopsis

I've been always wondering why Toronto doesn't have good subway map that can help people, especially newcomers, with their routes.
So, I decided to write my own.

# Installation
```
git clone <repository_url>
npm install
```

### Dev Server

``ng serve``

### Build

``ng build --prod``
